Acesse http://localhost:8080/pacientes.

Resultado esperado: Deve abrir a página pacientes.html (provavelmente uma tabela vazia ou com dados do banco).

Acesse http://localhost:8080/pacientes/novo.

Resultado esperado: Deve abrir o formulário (form-paciente.html).

Preencha o formulário e clique em "Salvar".

Resultado esperado: Ele deve redirecionar de volta para a lista e mostrar o novo paciente cadastrado.
